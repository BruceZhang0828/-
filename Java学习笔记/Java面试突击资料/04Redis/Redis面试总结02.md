### Redis面试总结

#### 1. Redis数据结构以及使用场景分析

##### string

###### 简单动态字符串(SDS),可以保存文本数据,保存二进制数据。

###### 常用命令：

- set：设置值 
- get：获取
- strlen：获取长度
- exists ：判断key是不是有值
- decr：数字类型-1
- incr：数字类型+1
- setex：设置值的时候加上过期时间

###### 应用场景：

​	需要计数的场景，比如用户访问次数，热点文章点赞，转发数量。

##### list

###### list即是链表。是一个双向链表，支持反向查找和遍历更方便操作。

###### 常用命令：

rpush：从链表头插入

rpop：从链表头弹出

lpush：从链表尾插入

lpop：从链表尾弹出

lrange：按照下标进行遍历 0 -1可以遍历所有。

llen：求数组的长度

###### 应用场景：

发布订阅或者说消息队列，慢查询。
